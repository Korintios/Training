# Training

Conoce mi campo de entrenamiento en programación.  
Aquí documento mis soluciones a ejercicios de **Codewars**,  
explicando mi razonamiento, cómo lo resolvió la comunidad y qué aprendí en el proceso.  

📂 Organización:
- `langs/` → Lenguajes de programación.
- `8kyu/` `7kyu/` ... → Nivel de dificultad (entre mas pequeño es el numero es mas dificil).
- Cada kata tiene:
  - `my.js` → mi solución.
  - `best.js` → pruebas.
  - `README.md` → explicación, aprendizajes, comparaciones.

✍️ Objetivo: tener un historial público de mi progreso como desarrollador y mostrar cómo enfrento problemas.
